## Run Locally

Clone the project

```bash
  git clone https://github.com/Mati-Pereira/app-pokedex
```

Go to the project directory

```bash
  cd app-pokedex
```

Install dependencies

```bash
  npm install
```

Run in dev mode

```bash
  npm run dev
```

## Tech Stack

**Client:** React, Next.js, TailwindCSS, Typescript

**Server:** Node
